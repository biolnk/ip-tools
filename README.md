# IP Multitool - Advanced Network Utility Suite

![Python Version](https://img.shields.io/badge/python-3.7+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

A comprehensive network utility tool with 30+ features for IP analysis, network diagnostics, and security testing.

## ✨ Features

- **IP Information Lookup** - Geolocation, ISP, ASN data
- **Network Scanning** - Port scanner, network discovery
- **DNS Tools** - Lookup, reverse DNS, brute forcing
- **Security Tools** - SSL checker, threat intelligence
- **Network Diagnostics** - Ping, traceroute, packet loss
- **Utility Tools** - Subnet calculator, IP converter, and more
